# 코드 스타일

## 기준

이 디렉터리는 현재 문서 중심 저장소다.
코드가 다시 추가되더라도 아래 원칙을 먼저 따른다.

## 원칙

### 1. 워크플로우를 재구현하지 않는다

검증된 ComfyUI 워크플로우에 이미 있는 영상 생성 로직을
별도 런타임으로 다시 복제하지 않는다.

### 2. 오케스트레이션만 작성한다

새 코드는 아래 역할에만 한정한다.

- 워크플로우 실행 호출
- 산출물 정리
- MMAudio 특징 추출
- 음악 생성 입력 변환
- ACE-Step 실행 정리

### 3. 문서와 구조를 먼저 맞춘다

새 코드를 쓰기 전에
어떤 워크플로우를 기준으로 삼는지,
어디까지를 AOG가 맡는지 문서에 먼저 명시한다.
